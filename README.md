# 🎮 PlayPass - Digital Game Top-Up Platform

PlayPass adalah platform *top-up* game digital yang dirancang untuk memberikan pengalaman transaksi yang cepat, aman, dan tanpa ribet bagi para *gamer*. Proyek ini menampilkan antarmuka web modern dengan alur kerja pengguna (UI/UX) yang mulus, mulai dari pemilihan game, keranjang belanja, hingga sistem cetak nota otomatis.

## ✨ Fitur Utama

**Sisi Pengguna (User):**
* **Katalog Interaktif:** Menampilkan berbagai game populer (Mobile Legends, Valorant, PUBG Mobile, Genshin Impact, dll).
* **Pencarian Pintar (*Live Search*):** Fitur pencarian *real-time* untuk menemukan produk atau game dengan cepat.
* **Histori & Manajemen Transaksi:** Pelacakan status pesanan (Berhasil, Menunggu, Gagal) yang rapi, lengkap dengan fitur unduh nota/invoice PDF.
* **Autentikasi Dinamis:** Antarmuka Login, Register, dan Lupa Password dengan simulasi verifikasi OTP.

**Sisi Administrator (Admin):**
* **Kelola Transaksi:** Dasbor khusus untuk memvalidasi bukti pembayaran, menerima pesanan, atau menolak transaksi.
* **Detail Investigasi:** Modal *pop-up* responsif untuk melihat detail pengguna, User ID tujuan, dan log struk secara mendalam.
* **Manajemen Sistem:** Antarmuka terstruktur untuk mengelola Produk, Pengguna, dan Artikel Promo.

## 🛠️ Tech Stack & Environment

Proyek ini dibangun menggunakan teknologi berikut:
* **Frontend:** HTML5, CSS3 (Custom Variables & Flexbox/Grid), Vanilla JavaScript.
* **Backend & Database Plan:** Sangat kompatibel untuk diintegrasikan dengan **Native PHP** maupun *framework* **Laravel**, dipadukan dengan MySQL.
* **Development Environment:** Direkomendasikan menggunakan **Laragon** sebagai *local server* untuk manajemen *database* dan *routing* yang lebih cepat dan ringan.
